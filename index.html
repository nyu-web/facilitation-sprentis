What?

  • Feature of CSS animation module
  • From traditional animation
  • Introduced in CSS3

Why?

  • Adobe Flash is outdated
  • JS can be complicated*
  • More functions than CSS transition
  • Written right in the CSS

Movement

Color

Change shapes

  • Styles
